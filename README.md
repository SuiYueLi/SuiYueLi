## 关于应用

这是一个PWA日历应用，是**华夏岁月历**的程序实现。

华夏岁月历由夏历增补节历而成，历法及相关概念与术语详见：[江俊.《历象授时·传承及补充——华夏岁月历》.[链接待补]]()，或参考本应用源文件中的 JieLi.js、XiaLi.js。

本应用以节历为主历法，以夏历为副历法，并附注干支、西历、节庆民俗等附加信息。

本应用提供简单的笔记功能。笔记存储在浏览器存储内，环境支持时（如 Windows + Edge）可同时指定本地存储文件。

笔记不会上传网络。如果没有指定本地存储文件，清除浏览器缓存、卸载浏览器将导致笔记及个人设置**数据丢失**，建议经常导出备份。

## 实用警告

本应用之主历法——节历，并非一种公共通行历法。

**切勿实用于现实日程，以免因混淆现行历法而误事！**

## 已知问题

- 安卓设备的物理返回键或只能回退一层页面，第二次会导致应用退出。
	- 权宜之计：优先使用应用内的返回按钮。
- Windows 下手动更新容易提示“更新失败”。
	- 权宜之计：再检查一下版本号，通常已更新成功。
- Windows 版 Edge 浏览器中，上下拖动日历格切换节月的滚动动画可能出现预期外的回滚视觉效果。
	- 权宜之计：反正节月是能顺利切换的。
