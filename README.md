# 关于

## 关于应用

这是一个PWA日历应用，是**华夏岁月历**的程序实现。

华夏岁月历由夏历结合节历而成。**节历**是一种基于二十四气的太阳历。

关于华夏岁月历的详细规则及相关的华夏历法要点，已撰写为一篇完整论文，预计将上传预印本平台。待上传后补充链接。

本应用以节历为主历法，以夏历为副历法，并附注西历、干支、节庆民俗等附加信息。

本应用提供简单的笔记功能。笔记存储在浏览器存储内，环境支持时（如Windows+Edge）可同时指定本地存储文件。

笔记不会上传网络。如果没有指定本地存储文件，清除浏览器缓存、卸载浏览器将导致笔记（及个人设置）**数据丢失**，建议经常导出备份。



日历编排、日期信息等历法实现程序均为手工编写；页面呈现则通过AI辅助完成（TRAR CN + GML-5.1）。

## 实用警告

本应用之主历法——节历，并非一种公共通行历法。

**切勿实用于现实日程，以免因混淆现行历法而误事！**
