可以安装的网页

这是一个渐进式网络应用（PWA）。

渐进式网络应用可以像普通网页一样添加到浏览器收藏夹/书签；当系统及浏览器环境支持时，也可以作为应用安装到本地系统。

安装后，应用依托于安装所用浏览器的引擎、存储空间，在没有地址栏的独立窗口中运行，并支持离线访问。

本应用未经压缩的总体积仅 6.5MB，一般使用下安装与否区别不大。如果要使用笔记功能，请务必先尝试导出节庆民俗列表或每年重复日列表，确保数据在当前浏览器中可正常导出。

安装环境

  • 在Windows系统下，用Edge浏览器打开PWA网页，地址栏会出现安装图标，如上图所示。
  • 本应用内置环境检测，当浏览器环境支持安装时，汉堡菜单左侧将显示“安装到本地”，点击即可执行安装。
  • Windows下安装PWA应用通常用Chromium系浏览器，即Chrome、Edge等；Firefox浏览器需要扩展支持。
  • 国产安卓系统的官方浏览器或常见流行浏览器不一定支持安装PWA，相关文件功能可能受限或需经曲折授权。建议使用Edge浏览器（需获取“桌面快捷方式”权限）。
  • macOS 14+系统的Safari浏览器（17+）支持安装PWA，但功能支持不如Chromium系。
  • iOS Safari浏览器支持安装PWA，但功能可能受限或不稳定；iOS 16.4+也可以用Chromium系，但功能支持不如Safari。

提示

  • 安装后应用数据存储在浏览器存储内。如果没有指定本地存储文件，清除浏览器缓存、卸载浏览器将导致笔记及个人设置数据丢失，建议经常导出备份。
  • 部分浏览器需先访问网站并停留片刻，安装选项才会出现。
  • 安装在Windows系统后，可以像普通软件一样卸载（但可能不会出现在传统控制面板的“程序和功能”列表中）。
  • Android系统下，安装动作如果无后续响应或跳转到浏览器的应用信息，意味着浏览器未获得在主屏幕上添加图标的权限，需要在应用权限管理中允许“桌面快捷方式”（可能要打开“其他权限”或“所有权限”才能找到）。
  • 安装在Android后，要移除只需删除图标，“卸载”可能会卸掉安装用的浏览器。

下面是AI写的各主流系统、浏览器环境下的安装说明，细节可能过时或不准确，仅供参考。

主流系统安装说明（AI版）

Android

Chrome

  1. 打开浏览器菜单（右上角 ⋮ ）；
  2. 点击「添加到主屏幕」或「安装应用」；
  3. 确认安装。

Edge

  1. 打开浏览器菜单（右上角 ⋯ ）；
  2. 点击「添加到手机」或「安装应用」；
  3. 确认安装。

iOS / iPadOS

Safari

  1. 点击底部分享按钮 ⬆；
  2. 向下滚动，点击「添加到主屏幕」；
  3. 点击「添加」确认。

Chrome (iOS 16.4+)

  1. 打开浏览器菜单（右上角 ⋮ ）；
  2. 点击「添加到主屏幕」；
  3. 确认添加。

Windows

Chrome

  1. 点击地址栏右侧安装图标 ⊕，或打开菜单（右上角 ⋮ ）→「安装华夏岁月历」；
  2. 点击「安装」确认。

Edge

  1. 点击地址栏右侧安装图标 ⊕，或打开菜单（右上角 ⋯ ）→「应用」→「将此站点作为应用安装」；
  2. 点击「安装」确认。

macOS

Chrome

  1. 点击地址栏右侧安装图标 ⊕，或打开菜单（右上角 ⋮ ）→「安装华夏岁月历」；
  2. 点击「安装」确认。

Edge

  1. 点击地址栏右侧安装图标 ⊕，或打开菜单（右上角 ⋯ ）→「应用」→「将此站点作为应用安装」；
  2. 点击「安装」确认。

Safari

macOS 版 Safari 对 PWA 的支持不如 Chromium 系浏览器完善，建议使用 Chrome 或 Edge。

Linux

Chrome / Chromium

  1. 点击地址栏右侧安装图标 ⊕，或打开菜单 →「安装华夏岁月历」；
  2. 点击「安装」确认。
